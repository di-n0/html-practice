css의 기본 문법

html과 css는 언어가 다르므로 어디서부터 어디까지 css인지 인식시켜줘야된다.
첫번째로 ＜style> ＜/style> 이란 태그이다. 두번째는 style 속성을 쓰는것
＜a href="xx.html" style="color:red">
태그 안에 style 이란 속성을 사용하게 되면 뒤에 나오는 명령어는 css 언어로 인식한다는 약속이 되어있음, 따라서 태그안 스타일 속성이 포함된 태그를 css언어로 표현할것이라는걸 말함
＜style>

a{

color:black;

}


＜/style>

반대로 ＜style>＜/style>란 태그는 css효과가 들어와도 어디에 적용된다는 코드를(보라색) 추가해줘야 작동. 보라색 코드 부분을 선택자(selector) 라고 표현하고,
선택자에게 (괄호안) 어떻게 변하게 해라 라는걸 선언(declaration)이라 함
괄호안에 변화를 주고싶은 항목을(빨간색) 속성(property) 이라 하고 변화될값을(녹색) value라고 한다
효과를 지정한 후에는 ;끝맺음을 해야됨
